⚡ HTML Master Editor 拡張版マニュアル

🖥 デスクトップアプリ 12言語対応拡張版 PyQt6 ベース・オフライン完全動作

🔍 概要・特徴

HTML Master Editor 拡張版は、HTMLをはじめとする12種類のファイルを 編集できるデスクトップアプリケーションです。 Python と PyQt6 で動作し、インターネット接続・サーバー不要で 単体起動できます。

主な特徴

  • ✅ 12言語対応シンタックスハイライト
  • ✅ 言語自動判別（ファイル拡張子で自動切り替え）
  • ✅ リアルタイムプレビュー （HTML はレンダリング、その他はテキスト表示）
  • ✅ 12言語対応構文チェック
  • ✅ HTMLコード補完 （Ctrl+Space / 2文字入力で自動表示）
  • ✅ 自動閉じタグ・オートインデント （HTML / XML）
  • ✅ 12言語対応ドキュメント構造ナビゲーター
  • ✅ 外部ファイル化（疎結合化） （HTMLのCSS/JSを別ファイルに切り出し）
  • ✅ プレビュー右クリック→ソースジャンプ
  • ✅ 別ウィンドウプレビュー
  • ✅ Ctrl+ホイールでフォントサイズ変更
  • ✅ 完全オフライン動作

👁 プレビュー機能

HTMLプレビュー

HTMLファイルを編集中は右パネルにリアルタイムでレンダリング結果が表示されます。 コードを編集すると約1.2秒後に自動更新されます。 ローカルファイルを開いている場合は同じフォルダの画像・CSSも正しく参照されます。

その他言語のプレビュー

HTML以外のファイルはダークテーマのテキストビューで表示されます。 右上に言語バッジ（例：Python）が表示されます。 コードの変更は約1.2秒後に自動反映されます。

別ウィンドウプレビュー

🗗 別ウィンドウ ボタンまたは F7 で 独立ウィンドウが開きます。

  • コードを編集すると別ウィンドウも自動更新されます
  • 別ウィンドウ内でも 🖥 📱 📟 ボタンで表示幅を変更できます
  • 別ウィンドウを閉じると埋め込みプレビューに自動で戻ります
  • 🗗 埋め込みに戻す ボタンでも戻せます

プレビュー右クリック→ソースジャンプ

プレビュー内の要素を右クリックして 🔍 View Page Source を選ぶと、 エディタの該当行にカーソルが移動します。

操作手順

  1. プレビュー画面内の編集したい要素を右クリック
  2. コンテキストメニューから 🔍 View Page Source を選択
  3. エディタが該当行にジャンプする
  4. ステータスバーに ジャンプ: <タグ名> → Line XX (score=XX) と表示される
💡 ジャンプの精度： id属性・class属性・src/href属性・テキスト内容をスコアリングして 最も一致度の高い行を特定します。 score の値が高いほど精度が高くなります。
⚠️ ソースジャンプは HTMLファイルのみ対応しています。 その他の言語ファイルでは動作しません。

💾 ファイル操作

ファイルを開く

  1. 📂 開く ボタンまたは Ctrl+O
  2. ファイル選択ダイアログから対象ファイルを選択
  3. 拡張子に応じて自動的に言語モードが切り替わる
💡 ファイルダイアログでは「すべての対応ファイル」を選択すると 12種類すべての拡張子が表示されます。 言語別にフィルタリングすることも可能です。

上書き保存

  1. 💾 保存 ボタンまたは Ctrl+S
  2. 既存ファイルを開いている場合は上書き保存
  3. 新規ファイルの場合は「名前を付けて保存」ダイアログが開く
💡 保存時のエンコーディングは UTF-8 固定です。

🔗 外部ファイル化（疎結合化）

HTMLファイル内の <style> ブロックや <script> ブロックを外部ファイルとして切り出す機能です。 HTMLファイルのみ対応しています。

操作手順

  1. 対象のHTMLファイルを保存済みの状態にする
  2. 🔗 外部ファイル化 ボタンをクリック
  3. ダイアログで切り出す対象と出力ファイル名を選択
  4. OK をクリック
  5. HTMLファイルと同じフォルダに .css / .js が生成される
  6. HTMLは自動的に <link> / <script src> に書き換えられる
⚠️ HTML以外のファイルを開いているときは 「外部ファイル化はHTMLファイルのみ対応しています」と表示されます。
⚠️ 未保存の状態では実行できません。先に保存してください。

✨ HTMLフォーマット

メニュー「ツール」→「HTMLフォーマット整形」または ツールバーの ✨ フォーマット ボタンで HTMLコードを自動整形します。 HTMLファイルのみ対応しています。

⚠️ HTML以外のファイルを開いているときは 「HTMLフォーマットはHTMLファイルのみ対応しています」と表示されます。

🛠 トラブル対応

症状対処方法
起動しない
  • Python 3.10以上がインストールされているか確認
  • pip install PyQt6 PyQt6-WebEngine lxml pygments を実行
  • コマンドプロンプトで直接実行してエラーメッセージを確認
色分けが切り替わらない
  • ファイルを「開く」で読み込んでいるか確認 （コピー貼り付けでは言語が切り替わりません）
  • 拡張子が対応リストにあるか確認
VBAファイルの色分けが効かない
  • 拡張子が .vba になっているか確認 （.bas 等は非対応）
CSVが文字化けする
  • ファイルが UTF-8 で保存されているか確認
  • Shift-JIS のCSVは一度UTF-8に変換してから開いてください
外部ファイル化ができない
  • HTMLファイルを開いているか確認
  • 先にファイルを保存（Ctrl+S）してから実行
プレビューが表示されない
  • PyQt6-WebEngine がインストールされているか確認
  • F5 で手動更新を試みる
右クリックジャンプが効かない
  • HTMLファイルを開いているか確認（他の言語では動作しません）
  • プレビューを F5 で更新してから再試行
画面レイアウトが崩れた
  • メニュー「表示」→「レイアウトをリセット」または F8
JSONナビゲーターに何も表示されない
  • JSONが配列（[ ]）の場合は件数のみ表示されます
  • JSONにエラーがある場合はナビゲーターに 「❌ JSON解析エラー」と表示されます
